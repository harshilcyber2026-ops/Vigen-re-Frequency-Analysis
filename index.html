Background

The Vigenère cipher uses a repeating keyword to shift each letter. Once called Le Chiffre Indéchiffrable, it was broken by Babbage & Kasiski using the key's repetition.


With a long ciphertext, statistical techniques reveal both the key length and each key letter.

How Vigenère Works

Cipher CVXPCO
